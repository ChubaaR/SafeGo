# SafeGo

SafeGo is a biometric-powered personal safety application designed to keep users safe during journeys and emergencies.

## Features

### 🔒 **Enhanced Security (Latest Updates)**
- **PopScope Protection**: All safety-critical dialogs now require biometric verification to dismiss
- **Journey Exit Protection**: Users cannot bypass safety measures during active journeys
- **SOS Alert Security**: Emergency alerts cannot be dismissed without proper authentication
- **Check-in Verification**: Interval check-ins require biometric confirmation for security

### 🚨 **Emergency Features**
- **SOS Alert System**: Quick emergency alerts to contacts with location sharing
- **Emergency Contacts Management**: Add and manage trusted emergency contacts
- **Live Location Sharing**: Real-time location tracking during emergencies
- **Biometric Authentication**: Face and fingerprint verification for security

### 🛣️ **Journey Tracking**
- **Journey Planning**: Set departure and destination with estimated time
- **Interval Check-ins**: Automatic safety check-ins during long journeys
- **Arrival Verification**: Secure confirmation when reaching destination
- **Background Monitoring**: Continuous safety monitoring even when app is backgrounded

### 📱 **Smart Notifications**
- **Check-in Reminders**: Automated prompts for safety check-ins
- **Emergency Alerts**: Push notifications for SOS situations
- **Journey Updates**: Status updates for emergency contacts
- **Missed Check-in Warnings**: Automatic alerts when check-ins are missed

## Security Features

SafeGo prioritizes user safety with comprehensive security measures:

- **Biometric Authentication**: All critical actions require face or fingerprint verification
- **PopScope Protection**: Prevents unauthorized exits from safety dialogs
- **Background Monitoring**: Continues safety monitoring even when app is minimized
- **Emergency Contact Integration**: Seamless communication with trusted contacts
- **Firebase Integration**: Secure cloud-based data storage and real-time updates
